## 💽 如何使用

1. 在[此处](https://github.com/StarWishsama/Comet-Bot/releases)下载最新版本
2. 使用 ```java -jar Comet本体路径``` 启动 Bot
3. 输入 /login 登录指定平台账号 (QQ, Telegram [WIP])
4. 提示登录成功后, 在机器人加入的群内发送 /help 就可开始了解 Comet 的各项功能了
